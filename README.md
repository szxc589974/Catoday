# Catoday

## 前言
身為貓咪鏟屎官，當然要做些跟貓咪相關的資訊，所以設計LineBot來讓各位鏟屎官更了解自己的貓咪!

## 環境
- windows
- python 3.6.8
- ngrok

## 技術
- 利用Google_API_Key更輕易獲取地圖資訊，從而計算附近動物醫院並選出評分高的

## 操作說明
- 基本操作
    -以下兩個指令皆可隨時輸入
        - `重新開始`
            -重設所有資訊
        - `fsm`
            - 傳回fsm圖片
            
- 架構圖
    1. 輸入`功能總覽`開始選擇以下四個功能
      - `貓咪肥胖指數`
          - 根據S.H.A.P.E.™ ，分數由A.非常瘦至G.重度肥胖
          - 由8個問題來測驗貓咪肥胖指數
      - `轉換年齡`
          - 幫助鏟屎官得知貓咪換算成人類歲數是幾歲
      - `想看療癒照片`
          - 可以得到隨機貓咪照片喔(可能不太療癒 :(
      - `尋找附近動物醫院`
          - 輸入自己的經緯度，就可以得到附近評分高的動物醫院
